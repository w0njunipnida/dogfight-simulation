# ✈️ Dogfight Flight Simulation

## 📌 소개
Three.js를 활용한 3D 비행 시뮬레이션 개인 프로젝트입니다.

## 🛠 기술 스택
- React
- Three.js
- Firebase
- Capacitor

## 🎮 주요 기능
- 3D 비행 시뮬레이션
- 사용자 입력 기반 조작
- Firebase 연동
